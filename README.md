## 💻 Installation & Local Development

### Prerequisites:
bash Python 3.9 or higher pip package manager
Git for version control Modern web browser (Chrome, Firefox, Safari, Edge)


### Quick Start - Local Installation:
bash

1. Clone the repository
git clone https://github.com/LeenaAdapakala17/ai-migration-platform.git cd ai-migration-platform

2. Create virtual environment
python -m venv venv source venv/bin/activate # On Windows: venv\Scripts\activate

3. Install dependencies
pip install -r requirements.txt

4. Run the application
streamlit run dashboard.py

5. Open browser to http://localhost:8501

### Development Environment Setup:
bash

Install development dependencies
pip install -r requirements-dev.txt

Run with hot reload for development
streamlit run dashboard.py --server.runOnSave true

Run tests
pytest tests/

Code formatting
black dashboard.py src/
